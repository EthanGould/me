Analog Clock

The classic analog clock, recreated with CSS and JavaScript!


A smooth radial gradient on the clock face, paried with a fine tuned bezier-curve provides a skeuomorphic effect as the clock ticks. The alignment of each clock hand is calculated in degrees based on the current moment in time.


This sample provides a realistic look and feel with each passing second.
